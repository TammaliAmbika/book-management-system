# Book Management System

A simple React-based Book Management System with CRUD operations, search, and filtering functionality.

## Live Demo
https://ambika-book-management.netlify.app

## GitHub Repository
https://github.com/TammaliAmbika/book-management-system

---

## Features

- View all books
- Add new books
- Edit existing books
- Delete books
- Search books by title or author
- Filter books by genre
- Responsive UI
- API integration with MockAPI

---

## Tech Stack

- React.js
- React Router DOM
- Axios
- Vite
- MockAPI

---
